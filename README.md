# SPKamatka

Премиальный сайт базы отдыха `spkamatka.ru` на Next.js.

## Локальный запуск

```bash
npm install
npm run dev
```

## Продакшен

```bash
npm run build
npm run start
```

## Деплой на VDS

Готовые файлы для деплоя лежат в папке `deploy`:

- `deploy/upload-vds.ps1`
- `deploy/setup-vds.sh`
- `deploy/nginx-sputnik.conf.example`
- `deploy/DEPLOY-VDS.md`
